📖 Instrucciones (README)

# productos-html-generator

Convierte archivos CSV o Excel en páginas HTML con formato de burbujas.

## Uso
1. Subí tu archivo index.html a un repositorio público en GitHub.
2. Activá GitHub Pages en Settings → Pages (Source: rama main, carpeta /root).
3. Abrí el link público que te da GitHub Pages.
4. Desde la página podés subir cualquier archivo .csv, .xls o .xlsx y ver los productos renderizados en burbujas.

## Notas
- El generador usa PapaParse para CSV y SheetJS para Excel.
- No requiere backend: todo corre en el navegador.
- Podés personalizar estilos en el bloque
